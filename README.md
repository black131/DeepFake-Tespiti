In this project, my teammate and I conducted a DTF analysis using AI and human-generated images. To do this, we combined three different deep learning models (ResNet50, SwimTransformer, Xception) to create a new hybrid model. We have completed our project, achieving 99% accuracy and an F1 score of 1. Our model might seem to have overfit, but it never actually overfit. The main reason for this is the proper use of the necessary libraries in our code. Additionally, all files in our dataset are in .jpeg format, and there is no missing or incorrect data.
